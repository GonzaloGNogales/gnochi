# SMPL Assets

SMPL files are not included in this release folder because they are distributed under a separate license.

To export OBJ meshes, place your SMPL files here, for example:

```text
SMPL_NEUTRAL.pkl
SMPL_MALE.pkl
SMPL_FEMALE.pkl
```

Inference can still run without these files, but it will export JSON pose/scene files only.
